python虚拟环境

Coco
2024-09-20 (Updated: 2024-09-20)
python

venv

创建虚拟环境

假设要在当前目录的.venv目录下创建虚拟环境，那么执行以下命令：

python3 -m venv .venv

激活虚拟环境

  • Linux

    source .venv/bin/activate

  • Windows

    .venv\Scripts\Activate.ps1

退出虚拟环境

退出虚拟环境，只需要在命令行/终端执行以下命令：

deactivate

删除虚拟环境

只需要删除虚拟环境文件夹即可。

virtualenv

安装virtualenv

执行以下命令：

pip install virtualenv

创建虚拟环境

假设要在当前目录的.venv目录下创建虚拟环境，那么执行以下命令：

virtualenv .venv

如果系统上安装了多个版本的python，可以指定使用特定的python版本创建虚拟环境：

virtualenv -p /usr/bin/python3.9 .venv

激活虚拟环境

  • Linux

    source .venv/bin/activate

  • Windows

    .venv\Scripts\activate

退出虚拟环境

执行以下命令：

deactivate

删除虚拟环境

只需要删除虚拟环境文件夹即可。

pipenv

安装pipenv

pip install pipenv

创建虚拟环境

在当前目录下执行以下命令：

pipenv install

安装包

pipenv install numpy

也可以通过requirements.txt安装：

pipenv install -r requirements.txt

使用虚拟环境

  • 方法一：直接运行python脚本

    pipenv run python main.py

  • 方法二：激活虚拟环境

    pipenv shell

删除虚拟环境

pipenv --rm

自动生成requirements.txt

打包整个环境下的包

pip freeze > requirements.txt

只打包项目中使用到的包

  1. 安装pipreqs

    pip install pipreqs

  2. 打包

    pipreqs .
