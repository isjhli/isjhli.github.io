343. ж•ҙж•°жӢҶеҲҶ

йўҳзӣ®й“ҫжҺҘпјҡ343. ж•ҙж•°жӢҶеҲҶ

ж–ҮжЎЈи®Іи§Јпјҡд»Јз ҒйҡҸжғіеҪ•

и§Ҷйў‘и®Іи§ЈпјҡеҠЁжҖҒи§„еҲ’пјҢжң¬йўҳе…ій”®еңЁдәҺзҗҶи§ЈйҖ’жҺЁе…¬ејҸпјҒ| LeetCodeпјҡ343. ж•ҙж•°жӢҶеҲҶ

жҖқи·Ҝ

  1. зЎ®е®ҡdpж•°з»„д»ҘеҸҠдёӢж Үзҡ„еҗ«д№үпјҡdp[i]иЎЁзӨәжӢҶеҲҶж•°еӯ—iпјҢеҸҜд»Ҙеҫ—еҲ°зҡ„жңҖеӨ§д№ҳз§Ҝдёәdp[i]
  2. зЎ®е®ҡйҖ’жҺЁе…¬ејҸпјҡdp[i] = max(dp[i], max(j * (i - j), dp[j] * (i - j))), where j = 1-i-1
  3. dpж•°з»„еҰӮдҪ•еҲқе§ӢеҢ–пјҡdp[2] = 1
  4. йҒҚеҺҶйЎәеәҸпјҡд»ҺеүҚеҗ‘еҗҺйҒҚеҺҶ

96.дёҚеҗҢзҡ„дәҢеҸүжҗңзҙўж ‘

йўҳзӣ®й“ҫжҺҘпјҡ96.дёҚеҗҢзҡ„дәҢеҸүжҗңзҙўж ‘

ж–ҮжЎЈи®Іи§Јпјҡд»Јз ҒйҡҸжғіеҪ•

и§Ҷйў‘и®Іи§ЈпјҡеҠЁжҖҒи§„еҲ’жүҫеҲ°еӯҗзҠ¶жҖҒд№Ӣй—ҙзҡ„е…ізі»еҫҲйҮҚиҰҒпјҒ| LeetCodeпјҡ96.дёҚеҗҢзҡ„дәҢеҸүжҗңзҙўж ‘

жҖқи·Ҝ

  1. зЎ®е®ҡdpж•°з»„д»ҘеҸҠдёӢж Үзҡ„еҗ«д№үпјҡdp[i]иЎЁзӨәiдёӘз»“зӮ№з»„жҲҗзҡ„дәҢеҸүжҗңзҙўж ‘зҡ„дёӘж•°дёәdp[i]
  2. зЎ®е®ҡйҖ’жҺЁе…¬ејҸпјҡdp[i] += dp[i - j] * dp[j - 1], where j = 1 - iпјҢд»ҘжҜҸдёӘе…ғзҙ jдёәж №з»“зӮ№пјҢе·Ұиҫ№зҡ„е…ғзҙ е°ҸдәҺjпјҢжңүj - 1дёӘпјҢеҸіиҫ№зҡ„е…ғзҙ еӨ§дәҺjпјҢжңүi - jдёӘпјҢдёӨиҫ№еҗ„з»„жҲҗдёҖдёӘдәҢеҸүжҗңзҙўж ‘гҖӮ
  3. dpж•°з»„еҰӮдҪ•еҲқе§ӢеҢ–пјҡdp[0] = 1
  4. йҒҚеҺҶйЎәеәҸпјҡд»ҺеүҚеҗ‘еҗҺйҒҚеҺҶ
